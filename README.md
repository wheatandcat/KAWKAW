# カウカウ (KAWKAW) - 架空ショッピングサイト

■ URL

- https://kawkaw.app/

## 主な機能

### 商品閲覧

- **ホーム画面**: 118種類の架空商品からランダムに20商品を表示。シャッフルボタンで入れ替え可能
- **カテゴリフィルター**: テクノロジー、ファッション、フード、ホーム、スポーツ、エンタメの6カテゴリ
- **商品検索**: ヘッダーの検索バーから商品名・説明・カテゴリで検索
- **商品詳細**: 星評価、割引率、架空の配送・保証情報、共有ボタン付き

### 特集ページ

- **本日のお得情報** (`/deals`): 日替わりの目玉商品、カテゴリ別お得情報、スタッフのおすすめ
- **タイムセール** (`/timesale`): カウントダウンタイマー付きの時間限定セール。売切れ進捗バー表示
- **新着アイテム** (`/new-arrivals`): 本日入荷・今週の新着・最近の入荷に分けて表示。NEW/HOTバッジ付き
- **ランキング** (`/ranking`): 人気順・評価順・レビュー数順・割引率順の4つのソートモード

### ショッピング機能

- **カート** (`/cart`): 商品の追加・数量変更・削除。localStorageで永続化
- **チェックアウト**: 架空の注文完了画面（実際の決済なし）
- **注文履歴** (`/orders`): 過去の架空注文を一覧表示

### レビュー機能

- 商品詳細画面からレビューを投稿可能（ニックネーム・星評価・コメント）
- PostgreSQLデータベースに保存され、全ユーザー間で共有

### UI/UX

- レスポンシブデザイン（スマホ・タブレット・PC対応）
- スマホ用ハンバーガーメニューに全ページへの導線を完備
- ダークモード対応
- 日本語UI
- 共有ボタン（スマホ: OS共有シート / PC: URLクリップボードコピー）

## 開発

```bash
npm run dev          # 開発サーバー起動
npm run db:push      # DBスキーマ反映
```

## 注意事項

このサイトで販売されている商品はすべて架空です。実際の商品は存在せず、決済も発生しません。買い物欲を安全に満たすためのシミュレーションサービスです。
